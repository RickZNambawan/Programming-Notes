** TO CREATE A REMOTE REPOSITORY **
-- Generate ssh-key first so that it will never ask you again to open your account --

* git --version 

* git config --global user.name "name here"
* git config --global user.email "email here"

* mkdir (name of the folder)

* git init

* touch .gitignore

* git remote add origin (URL of your github account)

* git push -u origin (name of your branch)

* touch README.md

* git add -A

* git commit -m "message here"

* git pull origin master

* git push origin master



** OR TO CLONE EXISTING REPOSITORY **
-- Generate ssh-key first so that it will never ask you again to open your account --

* git config --global user.name "name here"
* git config --global user.email "email here"

* mkdir (name of the folder)

* git clone (URL remote or Directory path for local)
